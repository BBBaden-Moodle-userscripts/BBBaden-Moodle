## all Userscript Libraries

### MoodleThemeDetector
Moodle Theme Detector is a library designed to identify the currently applied theme in Moodle. This library simplifies the process of theme detection, making it useful for userscripts or plugins that may need to adapt their behavior based on the active theme.

[GitHub](https://github.com/BBBaden-Moodle-userscripts/MoodleThemeDetector)


### potentialPointsExtractLib
potentialPointsExtractLib is a library utilized in the [potential-Points](https://github.com/BBBaden-Moodle-userscripts/potential-Points) userscript for Moodle. This library specializes in converting the user evaluation table into a 2D array, providing developers with a structured data format for further processing.

[GitHub](https://github.com/BBBaden-Moodle-userscripts/potentialPointsExtractLib)
